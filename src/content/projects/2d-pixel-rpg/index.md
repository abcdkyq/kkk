# 2D 像素风 RPG

## 项目概述

像素风格的 2D 角色扮演游戏，包含对话系统、背包系统和回合制战斗。注重叙事体验和像素美术表现。

## 技术亮点

### 对话系统

基于节点图的可视化对话编辑器：

- 支持分支对话与条件判断
- 对话内容与逻辑分离，方便策划编辑
- 支持对话中触发事件（如获得物品、触发战斗）

### 背包与装备系统

完整的物品管理系统：

- 物品数据使用 ScriptableObject 管理
- 支持物品堆叠、排序、筛选
- 装备系统影响角色属性，实时计算面板数值

### 回合制战斗

经典的回合制战斗系统：

- 速度值决定行动顺序
- 技能树系统，支持主动技能和被动技能
- 战斗动画使用 Unity Animator 状态机驱动

## 开发周期

约 4 个月，独立完成程序与部分美术工作。

## 收获与反思

这个项目让我对游戏系统设计有了更深的理解，特别是如何设计可扩展的数据结构来支撑复杂的游戏系统。对话系统的开发过程中，我学会了如何用编辑器扩展来提升开发效率。
