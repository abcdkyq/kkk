# 多人在线射击原型

## 项目概述

基于 Unity Netcode for GameObjects 的多人在线射击游戏原型，实现了网络同步、射击判定和基础的匹配系统。

## 技术亮点

### 网络架构

采用客户端-服务器模型（Client-Server）：

- 使用 Unity Netcode for GameObjects 作为网络框架
- 客户端预测 + 服务器权威校验，减少延迟感
- NetworkVariable 同步关键游戏状态

### 射击系统

基于射线检测的射击判定：

- 客户端即时反馈（枪口火焰、弹痕）
- 服务端进行最终伤害判定
- 支持不同武器类型（步枪、霰弹枪）的弹道差异

### 房间匹配

简易的房间系统：

- 基于 Unity Relay 实现 NAT 穿透
- 房间创建、加入、退出的完整流程
- 简单的准备状态同步与游戏开始逻辑

## 开发周期

约 2 个月，担任主程序，与一名美术合作完成。

## 收获与反思

网络游戏开发的复杂度远超单机游戏。这个项目让我理解了网络同步的核心挑战，以及为什么"客户端预测 + 服务器校验"是 FPS 游戏的标准方案。调试网络问题的经验也非常宝贵。
